Elyra: Elegance. Speed. Simplicity.

Download Learn Source Code

⭐ Elegant Systems Language

  • Allows for low-level memory access, perfect for systems programming.
  • Features a borrow-checker and memory safety features, inspired by Rust.
    • No more dangling pointers.
    • No more null pointer dereferences.
    • No more use-after-free.
  • Strongly typed with type inferences and explicit casting rules.
  • Robust error handling by values -- no more exception throwing.

⭐ Swift Execution

  • Always compiled to machine code -- backed by LLVM's optimizations.
  • Painless C interop, inspired by Zig.
    • Compile your Elyra program to C.
    • C/C++ compiler included.
    • Easy cross-compilation.
  • Concurrency: Work In Progress.

⭐ High Level Simplicity

  • Clean and concise syntax to eliminate boilerplate.
  • Easy to learn, easy to use.
  • Compile time metaprogramming -- no macros, ever.
  • Object oriented principles including traits and inheritance.
  • Easy to use package manager and toolchain.
